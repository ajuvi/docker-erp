# docker-lampp

## Crear els contenidors
```
docker-compose up -d
```

## Eliminar els contenidors
```
docker-compose down
```

## Gestionar els contenidors

Mostrar els contenidors
```
docker container ls -a
```

Parar els contenidors
```
docker-compose stop
```

Iniciar els contenidors
```
docker-compose start
```
